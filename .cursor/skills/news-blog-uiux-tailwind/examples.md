# Examples

## Example 1: News Homepage

### Prompt

Generate a modern Tailwind news homepage with:
- Sticky header
- Hero section
- Featured story
- Latest stories grid
- Trending sidebar
- Newsletter CTA

### Expected output pattern

- Full HTML page code with Tailwind classes.
- Clear mobile-first layout.
- Semantic landmarks and accessible navigation.

## Example 2: Blog Listing Page

### Prompt

Generate a blog listing page for technology news with:
- Category filter chips
- Search input UI
- 2-column article cards on desktop
- Pagination area

### Expected output pattern

- Full HTML page code with reusable card pattern.
- Card metadata includes title, summary, category, date, read time.
- Hover/focus states on cards and links.

## Example 3: Full Article Detail Page

### Prompt

Generate a full blog detail page about energy market trends with:
- Breadcrumb
- Title + subtitle
- Author/date/read time
- Cover image
- Table of contents
- Long-form content with at least 4 sections
- Quote block and bullet list
- Sources section
- Related posts

### Expected output pattern

- Complete HTML page code with Tailwind.
- Strong reading experience (`max-w-3xl`, comfortable leading, heading hierarchy).
- Includes all major article sections, not only intro text.
