# SEO Web

This is the Codex-readable mirror for `.cursor/skills/seo-web/SKILL.md`.
Read the Cursor skill when changing SEO-sensitive pages.

## Required SEO For New Article Pages

- Unique `<title>`, meta description, keywords, canonical URL, and robots tag.
- Open Graph and Twitter metadata with a topic-relevant image.
- JSON-LD `BlogPosting` or appropriate article schema.
- One clear `h1`, logical heading order, semantic `article`, `section`, and `nav` blocks.
- `thumbnailUrl`, social images, JSON-LD image, and visible cover image must stay synchronized.
- Add or update `sitemap.xml` for new public detail pages.
- Use `rel="nofollow"` for affiliate links.

## Content Quality

Write descriptions that match the actual article. Do not reuse generic metadata from another post.
